# 🧬 Генерируемый онлайн-тест (10-11 класс)

Интерактивное веб-приложение для подготовки к экзаменам и проверке знаний. Сайт работает полностью в браузере, не требует серверной части (сугубо клиентский JavaScript) и отлично оптимизирован для работы как на ПК, так и на смартфонах.

---

## 🚀 Основные возможности

* **Динамическая генерация**: сайт автоматически выбирает 20 случайных вопросов из загруженного вами текстового документа.
* **Локальная библиотека (IndexedDB)**: загруженные файлы с тестами сохраняются прямо в памяти вашего устройства (телефона или компьютера). Вам не придется выбирать файл заново при каждом входе.
* **Умное автосохранение (Session Lock)**: прогресс прохождения теста защищен от случайного закрытия вкладки, блокировки экрана или отключения питания. При повторном открытии вы продолжите с того же вопроса.
* **Защита от перезагрузок**: невозможно сбросить ошибочный ответ простым обновлением страницы — система фиксирует ваш выбор.
* **Адаптивная темная тема**: комфортное прохождение тестов в вечернее время. Выбранная тема оформления также сохраняется в памяти устройства.

---

## 📂 Требования к структуре файла с тестами

Для корректного распознавания вопросов скриптом, ваш `.txt` или `.md` файл должен иметь строго определенный формат. Вопрос должен начинаться с цифры, варианты ответов — с буквы с круглой скобкой, а правильный ответ помечается тегом `[ОТВЕТ: буква]` в конце строки.

**Пример правильной разметки:**
```text
1. Что является элементарной единицей эволюционного процесса?
а) отдельная особь
б) вид
в) подвид
г) популяция  [ОТВЕТ: г]
```

## 🛠️ Как запустить локально на Windows

1. Скачайте все файлы репозитория в одну папку:
   * `index.html` — разметка интерфейса.
   * `style.css` — стили оформления (с поддержкой темной темы).
   * `script.js` — логика работы, парсер файлов и база данных IndexedDB.
2. Дважды кликните по файлу `index.html`, чтобы открыть его в любом браузере.
3. Перетащите ваш файл с тестами на экран браузера или воспользуйтесь кнопкой выбора.

## 📱 Запуск онлайн

Благодаря хостингу **GitHub Pages** вы можете проходить тесты прямо со своего смартфона:
1. Перейдите по ссылке вашего [https://anton504540.github.io/bio-test/](https://w4yf4r3r.github.io/bio-test/) .
2. Нажмите кнопку **«Загрузить файл»** и укажите файл с тестами из памяти телефона (он автоматически добавится в вашу встроенную библиотеку).
3. При желании переключите тему на темную с помощью кнопки в верхнем углу экрана.
4. Добавьте страницу в закладки или вынесите как ярлык на «Главный экран» смартфона для быстрого доступа в формате PWA.
